# Format der zurückgegebenen JSON-Datei
``` json
{
    //Tagesordnungspunkt
    "title": "string",
    //Gekürzte Version des Titels
    "title_short": "string",
    //Redner
    "speaker": "string",
    //Zugehörigkeit: entweder Partei oder Amt, so wie im Protokoll vorhanden
    "affiliation": "string",
    //Datum der Rede
    "date": "Date in ISO 8601-1:2019 String Format",
    //Gesamter Inhalt der Rede
    "text": "string"
}
```
